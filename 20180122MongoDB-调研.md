## MongoDB:调研
> 动态数据结构
>> BSON(binary json)
>> 
   - bson 和 json对比有什么优势？
   	- 二进制查询效率高
   	- 还可以定义复杂的数据类型，如binary array
   - 易开发、易拓展
   	- 横向扩展
   - 连表(linking)
    - 没有mysql方便
   - ORM?
    - 可以不使用ORM
   - 事务
    - 不支持
    - 解决方案：
     - 两阶段事务，使用中间表记录数据流向，再进行数据操作
     - 数据操作之后应该需要回调修改中间表状态
     - 中间表需要自己去定义，自己配置，包括状态
   - MySQL 向 MongoDB 迁移：
    - 工具: mongify
    - 使用非常方便
